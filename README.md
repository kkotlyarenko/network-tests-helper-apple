# KC-tests-helper

Удобный офлайн-помощник по вопросам дисциплины «Компьютерные сети».

Приложение собрано сразу для двух платформ: iPhone и Apple Watch. Внутри — поиск по вопросам и ответам, фильтр только по задачам с картинками и просмотр иллюстраций из локальных ресурсов.

## Возможности

- поиск по вопросам и ответам;
- фильтр только по вопросам с изображениями;
- просмотр карточек с ответами и встроенными иллюстрациями;
- отдельная watchOS-версия с компактным интерфейсом;
- работа без сети: все данные лежат внутри проекта.

## Структура проекта

- `KC-tests-helper/` — основное iOS-приложение;
- `KC-tests-helper Watch App/` — версия для watchOS;
- `Resources/qa.json` — база вопросов;
- `Resources/images/` — изображения к вопросам и ответам.

## Требования

- macOS;
- Xcode 15 или новее.

## Как запустить

1. Откройте `KC-tests-helper.xcodeproj` в Xcode.
2. Выберите нужную схему:
	- `KC-tests-helper` для iOS;
	- `KC-tests-helper Watch App` для watchOS.
3. Нажмите `Run`.

## Примечания

- Приложение полностью работает с локальными данными.
- Для запуска не нужен интернет.
- Если добавляете новые вопросы или картинки, обновляйте `Resources/qa.json` и папку `Resources/images/`.
