# KC-tests-helper

Приложение-помощник с вопросами и иллюстрациями по дисциплине "Компьютерные сети".

## Что внутри

- iOS-приложение: KC-tests-helper/
- watchOS-приложение: KC-tests-helper Watch App/
- Данные вопросов: Resources/qa.json
- Иллюстрации: Resources/images/

## Требования

- macOS
- Xcode 15+

## Быстрый старт

1. Откройте KC-tests-helper.xcodeproj в Xcode.
2. Выберите нужную схему:
- KC-tests-helper для iOS
- KC-tests-helper Watch App для watchOS
3. Нажмите Run.

## Заметки

- Проект использует локальные ресурсы (JSON и изображения), сеть не требуется.
- Файл .gitignore уже настроен для типичных Xcode/macOS артефактов.
